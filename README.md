# `wordfreq-rs`
> A command-line tool written in Rust for counting word frequencies in text files or standard input.
> Supports customizable output formats (table, JSON, CSV), sorting options, and filtering by word length.

## Setup

```sh
cargo build
cargo run -- --help
```
